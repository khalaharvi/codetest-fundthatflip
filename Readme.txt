Fund That Flip Front-end Codetest

Instructions:

Create a front-end webapp for Borrower's profile utilizing the provided mocks as reference.
The page should be responsive on mobile and tablet, translating from desktop design provided.
The data you need is provided in data.json. The images are provided for you in the images folder. 

Use sass or scss to demonstrate your css knowledge.

Use javascript and the provided JSON (data.json) to display all the dynamic content (text) within each section.

Optionally use a JS framework such as React, Vue, or Backbone for data binding.

Use a bundler for CSS, JS compilation. 

Note: you do not need to create complex search functionality. A basic search is fine where you enter a borrower name "Joe borrower" and the information on the page changes.

Bonus: incorporate any animation to the design as you see fit. 

If you have any questions about the code-test feel free to email us dev@fundthatflip.com   

How to Submit:
Please provide a github repo link and a readme for how to run your app.
